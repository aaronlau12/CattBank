ChattBank

A Java web application developed as part of a Java III course. The application demonstrates database-driven web development using Servlets, JSP, JDBC, and object-oriented design principles.

Features
Customer login authentication
Database connectivity using JDBC
Customer and Account business objects
Account lookup functionality
CRUD operations against a relational database
Technologies
Java
Servlets
JSP
JDBC
SQL
Microsoft Access
NetBeans
Future Improvements
Additional account management features
Improved user interface
Enhanced validation and error handling
